### 題目： 多功能顯示器

### 實驗器材

- raspberry pi
- 七段顯示器
- 麵包板
- LED燈

### 實驗說明

這項實驗會透過 raspberry pi 內嵌 Ubuntu 來操控七段顯示器與 LED 模組，執行程式後會有四個選項可以選擇：
1. 顯示時間資訊 ( hour : minute )
2. 顯示時間資訊 ( minute : second)
3. 鬧鐘功能
4. 行人號誌燈號倒數功能
